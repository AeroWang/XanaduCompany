# web-plus
这是前端的项目，启动项目请先安装依赖包，依赖包在
node_modules目录，也就是说，你必须确保该目录存在才去启动项目。
## 项目相关的命令
### 安装依赖包
```bash
yarn install
```

### 开发模式下启动项目
```bash
yarn serve
```

### 打包到生产环境
``` bash
yarn build
```

### eslint语法修复
```bash
yarn lint
```

### 快速删除 node_modules 目录
```bash
yarn clear
```
